實作「YouTube API」套件在 Laravel 專案中使用（一）

Memo's Blog
2018-10-28
套件開發
Laravel, PHP, Packagist, YouTube, 套件開發, 程式設計

前言

本文實作一個可以讀取 YouTube API 的套件。

專案目錄

專案目錄如下：

使用 YouTube API

YouTube API 提供詳細的文件供開發者使用。

申請 API Key

首先到 Google 開發者平台建立專案，並且取得 API Key。

安裝相依套件

建立 composer.json 檔。

安裝 Guzzle 及 PHPUnit 相依套件。

實作

在 src 資料夾中新增一個 Youtube.php 檔。

測試

新增一個 phpunit.xml 檔，並將 API Key 設為環境變數。

在 tests 資料夾新增一個 YoutubeTest.php 檔。

執行測試。

使用

在 example 資料夾新增一個 index.php 檔。
