еүҚиЁҖ

жң¬ж–ҮзӮәгҖҢжҲҗзӮә Python ж•ёж“ҡеҲҶжһҗйҒ”дәәзҡ„з¬¬дёҖе ӮиӘІгҖҚиӘІзЁӢзҡ„еӯёзҝ’зӯҶиЁҳгҖӮ

Pandas

Pandas жҳҜдёҖеҖӢеғҸжҳҜ Excel зҡ„ Python ж“ҙе……зЁӢејҸеә«гҖӮ

дҪҝз”Ёйӯ”иЎ“ж–№жі•жҹҘзңӢиіҮж–ҷеӨҫе…§е®№гҖӮ

и®ҖеҸ–дёҖеҖӢ CSV жӘ”гҖӮ

еҸ–еҫ—еүҚдә”еҲ—иіҮж–ҷгҖӮ

еҸ–еҫ—ж•ҙиЎҢгҖҢеңӢж–ҮгҖҚзҡ„иіҮж–ҷгҖӮ

е°Үж•ҙиЎҢгҖҢеңӢж–ҮгҖҚзҡ„иіҮж–ҷж”ҫйҖІйҷЈеҲ—гҖӮ

дҪҝз”Ё mean() еҮҪж•ёеҸ–еҫ—гҖҢеңӢж–ҮгҖҚзҡ„е№іеқҮеҖӨгҖӮ

дҪҝз”Ё std() еҮҪж•ёеҸ–еҫ—гҖҢеңӢж–ҮгҖҚзҡ„жЁҷжә–е·®гҖӮ

дҪҝз”Ё plot() еҮҪж•ёз•«еҮәжҠҳз·ҡең–гҖӮ

дҪҝз”Ё hist() еҮҪж•ёз•«еҮәзӣҙж–№ең–гҖӮ

  • grid еҸғж•ёд»ЈиЎЁжҳҜеҗҰжңүз¶Іж јгҖӮ

еҹәжң¬ж•ёж“ҡеҲҶжһҗ

дҪҝз”Ё describe() еҮҪж•ёз®—еҮәеёёз”Ёзҡ„еҹәжң¬зөұиЁҲж•ёж“ҡгҖӮ

дҪҝз”Ё corr() еҮҪж•ёз®—еҮәеҗ„з§‘д№Ӣй–“зҡ„зӣёй—ңдҝӮж•ёгҖӮ

дҪҝз”Ё corr() еҮҪж•ёз®—еҮәгҖҢеңӢж–ҮгҖҚиҲҮгҖҢж•ёеӯёгҖҚд№Ӣй–“зҡ„зӣёй—ңдҝӮж•ёгҖӮ

дҪҝз”Ё sum() еҮҪж•ёз®—еҮәзёҪзҙҡеҲҶпјҢдёҰж–°еўһеҲ°йҷЈеҲ—е…§гҖӮ

  • sum() еҮҪж•ёзҡ„ axis еҸғж•ёжҳҜ 1 д»ЈиЎЁе°ҮеҗҢдёҖеҲ—зӣёеҠ гҖӮ

ж–°еўһдёҖеҖӢеҠ ж¬ҠйҒҺзҡ„дё»з§‘жҲҗзёҫеҲ°йҷЈеҲ—е…§гҖӮ

дҪҝз”Ё sort_values() еҮҪж•ёе…Ҳе°ҚгҖҢзёҪзҙҡеҲҶгҖҚйҖІиЎҢжҺ’еәҸпјҢеҶҚе°ҚгҖҢдё»з§‘гҖҚйҖІиЎҢжҺ’еәҸгҖӮ

  • ascending еҸғж•ёзӮә False д»ЈиЎЁйҷҚеҶӘжҺ’еәҸгҖӮ

зө„иЈқ DataFrame зү©д»¶

е»әз«ӢдёҖеҖӢдәӮж•ёзҹ©йҷЈгҖӮ

е»әз«ӢдёҖеҖӢ DataFrame зү©д»¶гҖӮ

е»әз«Ӣз¬¬дәҢеҖӢ DataFrame зү©д»¶гҖӮ

еҗҲдҪөдәҢеҖӢ DataFrame зү©д»¶пјҢж–№еҗ‘жҳҜеҲ—зҡ„ж–№еҗ‘гҖӮ

йҮҚж–°з·ЁжҺ’зҙўеј•гҖӮ

еҗҲдҪөдәҢеҖӢ DataFrame зү©д»¶пјҢж–№еҗ‘жҳҜиЎҢзҡ„ж–№еҗ‘гҖӮ

зөұиЁҲ UFO зӣ®ж“Ҡж•ёйҮҸ

и®ҖеҸ–з·ҡдёҠ CSV жӘ”гҖӮ

дҪҝз”Ё groupby() еҮҪж•ёзҫӨзө„еҢ–зү№е®ҡж¬„дҪҚпјҢдёҰдё”иЁҲж•ёгҖӮ

д»Ҙ Time ж¬„дҪҚйҖІиЎҢжҺ’еәҸпјҢдёҰдё”еҜҰйҡӣж”№и®ҠеҺҹдҫҶзҡ„ df_state и®Ҡж•ёгҖӮ

з•«еҮәй•·жўқең–гҖӮ

е®ҡдҪҚзү№е®ҡиіҮж–ҷ

е»әз«ӢдёҖеҖӢ DataFrame зү©д»¶гҖӮ

еҲ—еҮә B иЎҢеӨ§ж–ј 0 зҡ„иіҮж–ҷгҖӮ

дҪҝз”Ё loc() еҮҪж•ёе®ҡдҪҚзү№е®ҡзҜ„еңҚзҡ„иіҮж–ҷгҖӮ

жӣҙж”№зү№е®ҡзҜ„еңҚзҡ„иіҮж–ҷгҖӮ

еҲҶжһҗиӮЎзҘЁиіҮж–ҷ

е…ҲеңЁ Anaconda зөӮз«Ҝж©ҹе®үиЈқ pandas-datareader еҘ—д»¶гҖӮ

еҰӮжһңеҮәзҸҫд»ҘдёӢйҢҜиӘӨпјҡ

еүҮзӣҙжҺҘеҲ°е„Іеӯҳеә«дёӢијүдёӢдҫҶйҮҚж–°е®үиЈқгҖӮ

и®ҖеҸ– Yahoo зү№е®ҡжңҹй–“ Apple е…¬еҸёзҡ„иӮЎзҘЁиіҮж–ҷгҖӮ

е°ҮйҖҷдә”е№ҙе…§зҡ„ж”¶зӣӨеғ№з•«еҮәдҫҶгҖӮ

дҪҝз”Ё diff() еҮҪејҸиЁҲз®—е ұй…¬зҺҮпјҢдёҰз•«еҮәең–дҫҶгҖӮ

дҪҝз”Ё rolling() еҮҪејҸд»ҘжҜҸ 20 еӨ©зӮәеҚҖй–“иЁҲз®—е№іеқҮеҖӨпјҢдёҰз•«еҮәең–дҫҶгҖӮ
