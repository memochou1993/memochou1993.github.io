隊列

隊列是 FIFO (First In, First Out) 的資料結構。

優先隊列

優先隊列中的每個項目都有各自的優先級，優先級最高的項目最先得到服務。

  • heapq 的 heappop() 函式會返回最小値項，因此把 priority 的値變為負値。
  • format() 函式指定的 !r 格式會對應 repr() 函数，能將物件轉換為字串形式。
