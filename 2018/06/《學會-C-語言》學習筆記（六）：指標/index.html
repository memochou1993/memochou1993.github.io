前言

本文為《學會 C 語言》一書的學習筆記。

指標

指標是 C 語言的低階程式處理能力，可以直接存取電腦的記憶體位址。

C 語言的指標是一種變數，只是變數內容不是常數値，而是變數的位址。

在 C 程式誤用指標導致的程式錯誤十分難除錯，需要加倍小心。
