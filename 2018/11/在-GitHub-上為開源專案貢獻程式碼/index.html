在 GitHub 上為開源專案貢獻程式碼

Memo's Blog
2018-11-24
Git Repository
GitHub, Version Control

前言

以 Laravel 後台管理系統 Voyager 專案為例。

步驟

  1. 將 Voyager 專案 Fork 到自己的儲存庫。
  2. 下載到本機。
  1. 貢獻程式碼。
  1. 推送程式碼

  1. 在拉取頁面點選 New pull request 發請拉取請求。

  2. 點選 Compare across forks 比較程式碼。

  1. 點選 Create pull request 建立拉取請求。

  2. 等待作者通知。
