「JavaScript 30」學習筆記（四）

Memo's Blog
2018-05-04
「JavaScript 30」Study Notes
JavaScript, Programming

前言

本文為「JavaScript 30」教學影片的學習筆記。

目標

練習對陣列進行排序或計算。

筆記

  1. 篩選出 16 世紀的發明家。
  • filter() 方法會建立一個符合條件判斷的新的陣列。
  • foo => (...) 是 ES6 的箭頭函式。
  1. 處理一個發名家的姓名陣列。
  • map() 方法會建立一個經過處理的新的陣列。
  1. 用年代最遠到最近來排列發明家的生日。

或用三元運算子表達。

  • sort() 用於對陣列的元素進行排序。
  • 當參數 a 大於 b 是 1 的話，表示升冪。
  1. 計算所有發明家的壽命的總和。
  • reduce() 方法用於累加。
  • 0 是 total 的預設値。
  1. 用發明家的壽命來排序。
  • lastGuy 大於 nextGuy 是 -1 的話，表示降冪。
  1. 找出列表「Boulevards in Paris」中所有含有「de」的街道名稱。
  • textContent 屬性可以獲得元素的文本内容。
  1. 用英文字母排序發明家的姓氏。
  • aLast 大於 bLast 是 1 的話，表示升冪。
  1. 計算物件的各別數量。
  • 如果陣列內沒有該項目，則從 0 開始新增。
  • {} 是預設的空物件。
