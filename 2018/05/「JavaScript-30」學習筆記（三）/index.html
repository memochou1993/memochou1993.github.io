「JavaScript 30」學習筆記（三）

Memo's Blog
2018-05-03
「JavaScript 30」Study Notes
JavaScript, Programming

前言

本文為「JavaScript 30」教學影片的學習筆記。

目標

製作調色盤和濾鏡的效果。

筆記

CSS 的部分，先在 :root 裡面命名變數。

  • 變數的開頭必須是兩個破折號（--）。

使用 var() 呼叫變數。

  • filter 可以做出濾鏡效果。
  • blur 是濾鏡效果的模糊。

JavaScript 的部分，先選取所有輸入框元素。

設定一個 handleUpdate() 方法。

  • dataset 可以取得自定義屬性的値。
  • document.documentElement 會回傳目前文件中的根元素，即 <html> 元素。
  • setProperty 可以設置元素的鍵値屬性。

設定每一個輸入框元素只要有改變或滑鼠移過就觸發 handleUpdate() 方法。
