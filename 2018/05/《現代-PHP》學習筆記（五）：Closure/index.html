閉包

閉包是個在創造時就封裝了內部狀態的函式，這個被封裝的狀態會一直被保存在閉包中，即使環境消失了。

閉包和匿名函式（沒有名稱的函式）理論上是不同的事，不過 PHP 將他們視為相同。
