將 Vue 專案部署至 GitHub Pages 服務

Memo's Blog
2021-04-24
Deployment
GitHub, GitHub Pages, JavaScript, Programming, Vue

做法

修改 vue.config.js 檔，將 publicPath 設置為以專案名稱為名的資料夾路徑。

新增 deploy.sh 部署腳本。

執行部署腳本。

最後，至專案的 GitHub Pages 頁面，將分支設置為 gh-pages。
