前言

簡介

Kubernetes 的 Service 有不同的類型，選擇 ClusterIP 時，可以透過叢集的內部 IP 暴露服務，但是服務只能夠在叢集內部被訪問。

回到虛擬機。

如果直接從叢集外部透過 Cluster IP 去存取服務，會無法存取。

因此 Cluster IP 只能在叢集內部的 Node 或 Pod 中被存取，不允許外部應用程式存取。
