з°Ўд»Ӣ

NSQ з”ЁдҫҶи§ЈжұәеҲҶж•ЈејҸиЁҠжҒҜеӮійҒһпјҢеңЁ NSQ дёӯжңүе…©еҖӢйҮҚиҰҒзҡ„е…ғзҙ пјҢдёҖеҖӢжҳҜгҖҢдё»йЎҢгҖҚпјҢеҸҰдёҖеҖӢжҳҜгҖҢй »йҒ“гҖҚгҖӮ

дёҖеҖӢдё»йЎҢеҸҜд»ҘжңүеӨҡеҖӢй »йҒ“пјҢеҰӮжһңжңү 3 еҖӢдёҚеҗҢзҡ„жңҚеӢҷзӣЈиҒҪеҗҢдёҖеҖӢдё»йЎҢпјҢжӯӨжҷӮе°ұжңғй–Ӣе•ҹ 3 еҖӢй »йҒ“гҖӮеҰӮжһңеҸӘжңүдёҖеҖӢжңҚеӢҷпјҢеүҮдё»йЎҢзҡ„жүҖжңүиЁҠжҒҜеүҮз”ұжӯӨжңҚеӢҷжүҖж¶ҲиІ»гҖӮ

жҜҸдёҖеҖӢй »йҒ“йғҪиғҪжҺҘж”¶еҲ°дёҖеҖӢдё»йЎҢдёӢжүҖжңүиЁҠжҒҜзҡ„еүҜжң¬гҖӮ

е®үиЈқ

дҪҝз”Ё brew е®үиЈқ nsq зӣёй—ңжҢҮд»ӨгҖӮ

дҪҝз”Ё

дҪҝз”Ё nsqlookupd жҢҮд»Өе•ҹеӢ•дёҖеҖӢ NSQ еҸўйӣҶгҖӮ

й–Ӣе•ҹеҸҰдёҖеҖӢзөӮз«ҜиҰ–зӘ—пјҢдҪҝз”Ё nsqd жҢҮд»Өе•ҹеӢ•дёҖеҖӢ NSQ жңҚеӢҷгҖӮ

еҶҚй–Ӣе•ҹеҸҰдёҖеҖӢзөӮз«ҜиҰ–зӘ—пјҢдҪҝз”Ё nsqadmin жҢҮд»Өе•ҹеӢ•дёҖеҖӢ NSQ еҸўйӣҶзҡ„ UI д»ӢйқўпјҢеҸҜд»ҘйҖҸйҒҺзҖҸиҰҪеҷЁиЁӘе•Ҹ http://localhost:4171/гҖӮ

еҶҚй–Ӣе•ҹеҸҰдёҖеҖӢзөӮз«ҜиҰ–зӘ—пјҢзҷјеёғдёҖеҖӢиЁҠжҒҜеҲ° test й »йҒ“гҖӮ

еҶҚй–Ӣе•ҹеҸҰдёҖеҖӢзөӮз«ҜиҰ–зӘ—пјҢе°Ү test й »йҒ“зҡ„иЁҠжҒҜжҢҒзәҢеҜ«е…Ҙ /tmp иіҮж–ҷеӨҫдёӯзҡ„ж—ҘиӘҢжӘ”гҖӮ

еҶҚй–Ӣе•ҹеҸҰдёҖеҖӢзөӮз«ҜиҰ–зӘ—пјҢзӣЈиҒҪж—ҘиӘҢжӘ”гҖӮ

еҸҜд»ҘзҷјеёғжӣҙеӨҡзҡ„иЁҠжҒҜеҲ° test й »йҒ“дҫҶйҖІиЎҢи§ҖеҜҹгҖӮ

еҸғиҖғиіҮж–ҷ
