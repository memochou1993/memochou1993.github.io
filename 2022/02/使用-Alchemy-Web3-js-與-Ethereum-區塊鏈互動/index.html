前言

本文參考 Alchemy 的範例進行實作，Alchemy 是一個區塊鏈開發者平台，能夠讓開發者訪問以太坊區塊鏈上的 API 端點，並且可以讀寫交易。

前置作業

註冊

首先到 Alchemy 註冊，並選擇 Ethereum 生態系。

創建應用程式

完成表單以創建應用程式，例如：

  • 團隊名稱：Memo’s Team
  • 應用程式名稱：Memo’s App
  • 網路：選擇 Rinkeby 網路（為 OpenSea 所使用的測試網路）

點選「VIEW KEY」按鈕，並將 API Key 複製起來。
