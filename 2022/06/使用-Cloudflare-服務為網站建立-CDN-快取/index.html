еҒҡжі•

йҰ–е…ҲпјҢеңЁ Cloudflare е»әз«ӢдёҖеҖӢеёіиҷҹгҖӮ

й»һйҒёгҖҢAdd a siteгҖҚж–°еўһдёҖеҖӢз¶Із«ҷпјҢијёе…Ҙз¶Із«ҷзҡ„з¶ІеҹҹеҗҚзЁұгҖӮ

Cloudflare жңғй–Ӣе§ӢжҺғжҸҸжӯӨз¶Із«ҷзҡ„жүҖжңү DNS зҙҖйҢ„гҖӮ

DNS

йҖІеҲ°еҹҹеҗҚиЁ»еҶҠжңҚеӢҷе•Ҷзҡ„еҫҢеҸ°пјҢй»һйҒё DNS з®ЎзҗҶпјҢиЁӯзҪ®з”ұ Cloudflare жҸҗдҫӣзҡ„еҗҚзЁұдјәжңҚеҷЁпјҡ

  • xxx.ns.cloudflare.com
  • yyy.ns.cloudflare.com

еӣһеҲ° Cloudflare иЁӯе®ҡ DNS зҙҖйҢ„пјҡ

Type Name Content Proxy Status TTL
A your-domain.com your-ip Proxied Auto
CNAME blog your-domain.com Proxied Auto

Caching

еүҚеҫҖгҖҢCache RulesгҖҚй ҒйқўпјҢе»әз«ӢиҰҸеүҮеҫҢпјҢжҢүдёӢгҖҢйғЁзҪІгҖҚжҢүйҲ•гҖӮ

Field Operator Value
Hostname contains your-domain.com

йҒӢз®—ејҸй җиҰҪеҰӮдёӢпјҡ

жӘўжҹҘ

еҸҜд»Ҙеҫһ CF-Cache-Status жЁҷй ӯжӘўжҹҘиіҮжәҗжҳҜеҗҰе·Іеҝ«еҸ–гҖӮд»ҘдёӢжҳҜжүҖжңүеҝ«еҸ–еӣһжҮүзӢҖж…Ӣпјҡ

вҖӢвҖӢHIT

иіҮжәҗеңЁ Cloudflare зҡ„еҝ«еҸ–дёӯжүҫеҲ°гҖӮ

вҖӢвҖӢMISS

иіҮжәҗжңӘеңЁ Cloudflare зҡ„еҝ«еҸ–дёӯжүҫеҲ°пјҢиҖҢжҳҜеҫһеҺҹе§Ӣз¶Ій ҒдјәжңҚеҷЁжҸҗдҫӣгҖӮ

вҖӢвҖӢNONE/UNKNOWN

Cloudflare з”ҹжҲҗдәҶдёҖеҖӢиЎЁзӨәи©ІиіҮз”ўдёҚйҒ©еҗҲйҖІиЎҢеҝ«еҸ–зҡ„еӣһжҮүгҖӮ

вҖӢвҖӢEXPIRED

иіҮжәҗеңЁ Cloudflare зҡ„еҝ«еҸ–дёӯжүҫеҲ°пјҢдҪҶе·ІйҒҺжңҹпјҢдёҰеҫһеҺҹе§Ӣз¶Ій ҒдјәжңҚеҷЁжҸҗдҫӣгҖӮ

вҖӢвҖӢSTALE

иіҮжәҗеҫһ Cloudflare зҡ„еҝ«еҸ–дёӯжҸҗдҫӣпјҢдҪҶе·ІйҒҺжңҹгҖӮCloudflare з„Ўжі•иҒҜз№«еҺҹе§ӢдҫҶжәҗд»ҘжӘўзҙўжӣҙж–°зҡ„иіҮжәҗгҖӮ

вҖӢвҖӢBYPASS

еҺҹе§ӢдјәжңҚеҷЁйҖҡйҒҺе°Ү Cache-Control жЁҷй ӯиЁӯзҪ®зӮә no-cacheгҖҒprivate жҲ– max-age=0 жҢҮзӨә Cloudflare з№һйҒҺеҝ«еҸ–пјҢеҚідҪҝ Cloudflare жңҖеҲқеёҢжңӣе°ҚиіҮз”ўйҖІиЎҢеҝ«еҸ–гҖӮ

вҖӢвҖӢREVALIDATED

иіҮжәҗеҫһ Cloudflare зҡ„еҝ«еҸ–дёӯжҸҗдҫӣпјҢдҪҶе·ІйҒҺжңҹгҖӮиіҮжәҗе·ІйҖҡйҒҺ If-Modified-Since жЁҷй ӯжҲ– If-None-Match жЁҷй ӯйҮҚж–°й©—иӯүгҖӮ

вҖӢвҖӢUPDATING

иіҮжәҗеҫһ Cloudflare зҡ„еҝ«еҸ–дёӯжҸҗдҫӣпјҢдҪҶе·ІйҒҺжңҹпјҢдҪҶеҺҹе§Ӣз¶Ій ҒдјәжңҚеҷЁжӯЈеңЁжӣҙж–°иіҮжәҗгҖӮUPDATING йҖҡеёёеғ…з”Ёж–јйқһеёёеҸ—жӯЎиҝҺзҡ„еҝ«еҸ–иіҮжәҗгҖӮ

вҖӢвҖӢDYNAMIC

Cloudflare дёҚиӘҚзӮәиіҮз”ўз¬ҰеҗҲеҝ«еҸ–жўқд»¶пјҢдё”еҝ«еҸ–иЁӯзҪ®дёӯдёҰжңӘжҳҺзўәжҢҮзӨә Cloudflare е°ҚиіҮз”ўйҖІиЎҢеҝ«еҸ–гҖӮж–јжҳҜеҫһеҺҹе§Ӣз¶Ій ҒдјәжңҚеҷЁи«ӢжұӮдәҶиіҮз”ўгҖӮ

еҸғиҖғиіҮж–ҷ
