使用 GitHub Actions 將 React 專案部署至 GitHub Pages 服務

Memo's Blog
2022-03-25
Deployment
GitHub, GitHub Actions, GitHub Pages, JavaScript, Programming, React

做法

首先，到專案的「Settings」頁面，將「Workflow permissions」設置為「Read and write permissions」。

在專案的 .github/workflows 資料夾新增 gh-pages.yaml 檔。

推送程式碼。
