在 macOS 上安裝 PHP 8.0

Memo's Blog
2022-03-07
Installation
PHP, Programming, macOS

移除舊版本

如果先前有使用 Homebrew 安裝過 PHP，可以使用 brew 指令移除。

手動刪除相關檔案。

安裝新版本

使用 brew 安裝最新版本。

啟動 PHP-FPM 服務。

關閉 PHP-FPM 服務。

重新啟動 PHP-FPM 服務。

修改環境變數

修改 .zshrc 檔：

  • bin 資料夾存放 PHP-CLI 的執行檔。
  • sbin 資料夾存放 PHP-FPM 的執行檔。

重新讀取環境變數。

查看 PHP-CLI 版本。

查看 PHP-FPM 版本。
