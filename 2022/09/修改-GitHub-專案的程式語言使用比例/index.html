修改 GitHub 專案的程式語言使用比例

Memo's Blog
2022-09-11
Git Repository
GitHub, Version Control

前言

GitHub 使用 linguist 來偵測專案使用的程式語言，透過設定，可以修改專案中程式語言的使用比例。

做法

在專案的根目錄新增 .gitattributes 檔。

修改 .gitattributes 檔，排除指定資料夾底下的所有檔案。
