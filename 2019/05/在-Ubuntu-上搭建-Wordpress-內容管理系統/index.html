在 Ubuntu 上搭建 Wordpress 內容管理系統

Memo's Blog
2019-05-12
Wordpress
Deployment, Linux, PHP, Ubuntu, Wordpress

環境

  • Ubuntu 18.04 LTS
  • Laradock

做法

啟動容器

建立資料庫。

下載 Wordpress。

設定權限。

在 ~/Laradock/nginx/sites 資料夾新增 wordpress.epoch.tw.conf 檔。

重新啟動 Nginx 容器。

前往 http://wordpress.xxx.com 瀏覽。

補充

如果為 Laradock 環境，資料庫的 host 欄位應設為 mysql。
