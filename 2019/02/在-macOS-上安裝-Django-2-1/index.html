在 macOS 上安裝 Django 2.1

Memo's Blog
2019-02-09
Django
Django, Programming, Python

安裝 Python

使用 Homebrew 安裝 Python 3。

添加環境變數至 ~/.bash_profile 檔。

重新加載啟動文件。

查看 Python 版本。

安裝 Pip

下載 get-pip.py 檔。

安裝 Pip3。

添加環境變數至 ~/.bash_profile 檔。

重新加載啟動文件。

查看 Pip 版本。

安裝 Virtualenvwrapper

使用 Pip 安裝 Virtualenvwrapper。

添加環境變數至 ~/.bash_profile 檔。

重新加載啟動文件。

建立虛擬環境。

列出可用的虛擬環境。

刪除指定的虛擬環境。

啟動指定的虛擬環境。

退出當前的虛擬環境。

安裝 Django

進入虛擬環境後，使用 Pip 安裝 Django。

查看 Django 版本。

使用 django-admin 工具建立一個專案。

啟動伺服器。

瀏覽網頁

前往 http://127.0.0.1:8000 瀏覽。
