在 Laravel 5.7 使用 Horizon 隊列管理工具

Memo's Blog
2019-02-22
Laravel
Horizon, Laravel, PHP, Programming

環境

  • Laradock

步驟

一般使用

啟動 Laradock。

建立專案。

修改 .env 檔。

安裝 laravel/horizon 套件。

執行安裝。

建立 failed-table 遷移檔。

執行遷移。

啟動 Horizon 服務。

前往 http://project.test/horizon 瀏覽。

Supervisord

如果要讓 Laradock 自動啟動 Horizon 服務，需要複製範本 laravel-horizon.conf.example 檔作為設定檔。

修改 project-horizon.conf 檔。

  • 參數 command 必須指向專案下的 artisan 腳本。

重新讀取設定檔。
