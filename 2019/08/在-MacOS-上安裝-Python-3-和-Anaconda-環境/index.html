在 macOS 上安裝 Python 3 和 Anaconda 環境

Memo's Blog
2019-08-24
Installation
Anaconda, Programming, Python

安裝環境

到官方網站下載 Anaconda 指令列安裝器。下載後，打開終端機執行安裝腳本，並指定安裝目錄為 ~/library/anaconda3。

將 conda 指令添加至環境變數。

取消開機時自動啟動虛擬環境。

安裝套件

使用 conda 指令安裝 keras，tensorflow 會一併安裝。

查看 keras 套件版本。

查看 tensorflow 套件版本。

打開筆記本

使用 jupyter 指令打開筆記本。
