在 macOS 上使用 Docker 建立 PHP 環境

Memo's Blog
2019-08-27
Deployment
Deployment, Docker, Nginx, PHP

步驟

建立專案。

拉取 richarvey/nginx-php-fpm 映像檔。

啟動容器。

  • 參數 --name 為容器名稱。
  • 參數 --restart 為自動重新啟動容器。
  • 參數 -p 為對外與對內的埠號。
  • 參數 -v 為對外與對內的映射目錄。

進入容器。

  • 參數 -i 會確保 STDIN 持續開啟。
  • 參數 -t 會分配一個虛擬終端。

在容器中修改 Nginx 設定。

修改 default.conf 檔的 root 參數。

重新啟動 Nginx 服務。

瀏覽網頁

前往 http://localhost:8081 瀏覽。
