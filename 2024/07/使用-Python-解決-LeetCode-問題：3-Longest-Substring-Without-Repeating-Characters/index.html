使用 Python 解決 LeetCode 問題：3. Longest Substring Without Repeating Characters

Memo's Blog
2024-07-27
Algorithm
Algorithm, LeetCode, Programming, Python

Description

Given a string s, find the length of the longest substring without repeating characters.

Solution

以下使用滑動視窗（Sliding Window）技巧，來找出字符串中最長的無重複字符子串。

滑動視窗很適合處理這類連續子資料的問題，使用兩個指針來表示當前窗口的左右邊界（left 和 right），並且動態調整窗口，來找到符合條件的子串。
