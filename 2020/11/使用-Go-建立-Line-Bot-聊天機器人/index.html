使用 Go 建立 LINE Bot 聊天機器人

Memo's Blog
2020-11-28
Others
Go, LINE, Programming, chatbot

建立頻道

首先，登入 LINE Developers 頁面，選擇 Messaging API 產品，建立一個 Channel。

建立專案

建立專案。

啟用 Go Modules。

下載 linebot 套件。

新增 main.go 檔：

新增 .env 檔：

  • CHANNEL_SECRET 環境變數填入 Basic settings 頁面的 Channel secret。
  • CHANNEL_ACCESS_TOKEN 環境變數填入 Messaging API 頁面的 Channel access token。

最後，將專案部署到主機。

設定

  1. 進到「Messaging API」頁面，設置應用程式的「Webhook URL」。

  1. 點選「Verify」按鈕。

  2. 將「Use webhook」功能開啟。

  3. 將「Auto-reply messages」和「Greeting messages」功能關閉。

聊天

進到「主頁」，點選「加入好友」，使用行動條碼加入好友。
