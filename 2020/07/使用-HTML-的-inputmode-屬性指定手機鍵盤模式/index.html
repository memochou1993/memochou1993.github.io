使用 HTML 的 inputmode 屬性指定手機鍵盤模式

Memo's Blog
2020-07-10
HTML
HTML, PWA, Programming, RWD

做法

設置為 numeric，可以顯示數字鍵盤。

設置為 decimal，可以顯示帶有小數點的數字鍵盤。

設置為 none，則完全不顯示鍵盤（可能使用客製化鍵盤）。
