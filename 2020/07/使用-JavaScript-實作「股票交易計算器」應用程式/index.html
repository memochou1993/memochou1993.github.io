前言

功能需求如下：

  1. 不要有按鈕，輸入完價格直接看到計算結果。
  2. 要可以查看前後五檔的成交價格，不需要重新輸入新的價格。
  3. 電腦版要可以使用上下鍵做價格調整，並且以正確的檔位做為升降單位。
  4. 手機版要可以使用數字鍵盤輸入價格。

避免花費太多時間在變數命名上，因此直接使用中文撰寫。
