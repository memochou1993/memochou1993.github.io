Memo's Blog
2020-02-09
其他

環境

做法

當讀取大型 XML 檔案時，應避免使用 ioutil.ReadAll() 和 xml.Unmarshal() 方法，而是使用 xml.Decoder.Token() 方法，來達到串流處理。
