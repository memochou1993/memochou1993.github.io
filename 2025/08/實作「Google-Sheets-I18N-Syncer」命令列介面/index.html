前置作業

首先要取得一個存取 Google Sheets API 的金鑰。步驟如下：

  1. 前往 Google Cloud。
  2. 在控制台輸入「Google Sheets API」，並啟用。
  3. 點選「IAM 與管理」頁籤，點選「服務帳戶」頁籤，建立一個服務帳戶。
  4. 點選建立好的服務帳戶，點選「金鑰」頁籤，建立一個 JSON 格式的金鑰。
  5. 建立一個試算表，與服務帳戶共用。
