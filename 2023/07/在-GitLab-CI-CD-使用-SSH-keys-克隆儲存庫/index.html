在 GitLab CI/CD 使用 SSH keys 克隆儲存庫

Memo's Blog
2023-07-13
GitLab
CI/CD, Deployment, GitLab, SSH

做法

首先建立一組 SSH 金鑰對。

在當前專案的 CI/CD 變數中，新增名為 SSH_PRIVATE_KEY 的環境變數，並且將值設定為 SSH 私鑰的內容。

在要被克隆的專案中，將 SSH 公鑰添加至該專案的 Deploy keys 中。

然後在當前專案新增 .gitlab-ci.yml 檔。

最後，將程式碼推送到儲存庫。
